### Getting Started
- Clone the Repo
- `git checkout [project]`

### Running each project
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
> Any additional details will be stated on the project's branch.
